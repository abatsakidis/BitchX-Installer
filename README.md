## Description ##

Bash script to install BitchX really quick

## How To ##

* chmod +x BitchX-Installer.sh
* ./BitchX-Installer.sh
<br>
And this will begin to install everything you need to run BitchX, then just type to run:
<br>
* BitchX


## Tested on ##

**OS**: Kali Linux <br>
**CPU**: Intel 2 Quad Q6600 (4) @ 2.400GHz <br>
**Memory**: 4085MB <br>
